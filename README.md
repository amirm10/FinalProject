# Overview

This repository contains the complete materials for my B.Sc. Final Project in Software Engineering.

The project presents an **Authorship Verification** system that determines whether two text samples were written by the same author.

The proposed solution combines modern Natural Language Processing and Deep Learning techniques, including:

- Siamese BERT
- CNN–BiLSTM
- Dynamic Time Warping (DTW)
- Isolation Forest
- Impostor Projection methodology

The system was evaluated using the **PAN20 Authorship Verification benchmark**.
