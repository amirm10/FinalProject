# Overview

Phase A focuses on the **research, analysis, and system design** of the proposed Authorship Verification framework.

The objective of this phase was to investigate existing approaches in authorship verification, identify their limitations, and design a complete deep-learning architecture that would later be implemented and evaluated in **Phase B**.

The proposed framework combines several modern NLP and machine learning techniques, including:

- **BERT** for contextual language representation.
- **Siamese Neural Networks** for pairwise authorship comparison.
- **CNN–BiLSTM** for stylistic feature extraction.
- **Dynamic Time Warping (DTW)** for style signal alignment.
- **Isolation Forest** for anomaly-based decision making.
- **Impostor Projection** methodology for improving robustness against writing style imitation.
