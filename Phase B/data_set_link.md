# External Resources

This project uses several external resources that are not fully stored in this GitHub repository due to file size limitations.

---

## 1. PAN20 Dataset

The project is based on the PAN20 Authorship Verification dataset.

Dataset (Hugging Face):
https://huggingface.co/datasets/swan07/authorship-verification/tree/main

Required CSV files:

- pan20_train.csv
- pan20_val.csv
- pan20_test.csv

---

## 2. Google Drive – Dataset and Trained Checkpoints

The complete dataset and the trained model checkpoints are available in the following Google Drive folder:

https://drive.google.com/drive/folders/1SZu5v-sBNRJD8prQ28FNRjB09SEb-GiS?usp=drive_link

The folder contains:

- data/
  - pan20_train.csv
  - pan20_val.csv
  - pan20_test.csv

- checkpoints/
  - best_model_final.pt
  - checkpoint_epoch_*.pt

These files must be downloaded before running the project.
